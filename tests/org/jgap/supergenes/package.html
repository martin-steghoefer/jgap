Provides methods for testing the Supergene. To test this method, we use the following task: given amount in pennies [1..99], express it in pennies, nickels (5 pennies), dimes (10 pennies) and quarters (25 pennies), additionally ensuring that numbers of pennies and nickels are both even or both odd. The Supergene, used in SupergeneTest, encloses pennies and nickels and is valid if this condition is satisfied. The alternative algorithm (withoutSupergeneTest) does exactly the same task without using supergenes.
